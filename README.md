# Hệ Thống Quản Lý Bệnh Viện

Ứng dụng quản lý bệnh viện được xây dựng nhằm hỗ trợ các hoạt động quản lý trong bệnh viện như quản lý bệnh nhân, bác sĩ, lịch hẹn khám bệnh, hồ sơ bệnh án và quản trị hệ thống.

---

## Chức Năng Chính

### Xác Thực & Phân Quyền
- Đăng nhập / đăng xuất

### Quản Lý Bệnh Nhân
- Thêm / sửa / xóa thông tin bệnh nhân
- Tìm kiếm bệnh nhân
- Xem lịch sử khám bệnh

### Quản Lý Bác Sĩ
- Quản lý thông tin bác sĩ
- Phân khoa/phòng ban
- Quản lý lịch làm việc

### Quản Lý Lịch Hẹn
- Đặt lịch khám bệnh
- Cập nhật trạng thái lịch hẹn
- Quản lý check-in bệnh nhân

### Hồ Sơ Bệnh Án
- Lưu thông tin chẩn đoán
- Quản lý điều trị
- Quản lý đơn thuốc

### Trang Quản Trị
- Thống kê hệ thống
- Quản lý người dùng
- Quản lý khoa/phòng

## Hướng Dẫn Cài Đặt

### 1. Clone repository

```bash
git clone https://github.com/Nozs002/Hospital_Management.git
```

### 2. Cấu hình database

Cầu hình database với workbench và mysql server
Mật khẩu và tên đăng nhập phải khớp với file connectMysql.java và JDBCUtil.java trong foler /src/connection

### 3. Chạy ứng dụng


Ứng dụng sẽ chạy tại chính IDE
